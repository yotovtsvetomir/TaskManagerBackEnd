# Back End
This project is a Django Rest Framework App backed by SQLite database

## Installing
- Make sure you have `python3` and `pip3` installed
- Create venv `python3 -m venv env`
- Activate venv `source env/bin/activate  # On Windows use env\Scripts\activate`
- Install the dependencies: `pip install -r requirements.txt`

## Running
- Start the app: `python manage.py runserver`
- By default the server starts on port 8000

## Testing
- Run tests: `python manage.py test`
